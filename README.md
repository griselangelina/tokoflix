This project was bootstrapped with Create React App.

Run below script to start the project in your local, with these steps:
### npm install 
### npm start

Runs the app in the development mode.
### Open http://localhost:3000 to view it in the browser.

This site provided 2 pages:

1. Homepage (in local: http://localhost:3000/) Showing all the Now Playing movie list 

2. Detail (in local: http://localhost:3000/Rumah) Showing the movie based on the movie id that chosen by user

Fitur:
1. Credit of User, There is credit of user amount that will be decreased every user buy the movie
2. Collection logo, If a movie already listed as collection then there will be a logo that show it's a user collection
3. User cannot buy the same movie
4. paging, there's paging for homepage movie list

Test the app in the development mode by running
### npm run test
